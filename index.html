Context

getContext

Get the standard canvas context (used for drawing)

canvas.getContext(contextType <String>, contextAttributes <Object> (Optional));
Returns context <RenderingContext>

getCurrentContext

Get canvas context used by Canvasimo (2d)

canvas.getCurrentContext();
Returns context <RenderingContext>

getCurrentContextType

Get the context type used by Canvasimo (2d, webgl, etc)

canvas.getCurrentContextType();
Returns contextType <String>

getContextAttributes

Get the context attributes used

canvas.getContextAttributes();
Returns contextAttributes <Object>
